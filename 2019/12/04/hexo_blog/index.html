Github Page + Hexo 搭建静态博客

Github Page 初始化

Key Generate

生成密钥并从.ssh目录下copy公钥(id_rsa.pub)

配置 Github SSH key

在Github仓库创建新的SSH KEY, 这样就可以向Github提交代码

创建 Github 仓库

用Github账号名做仓库的前缀(github_name.github.io)

Hexo Start

Install npm && node

需要安装npm和node.js环境(可能需要更新node版本)

Install Hexo

Generate init files

修改博客根目录_config.yml

修改_config.yml中deploy对应的内容，后续可push到对应的仓库

更换主题

我使用的主题是butterfly, 根据配置将主题pull到themes文件夹内,修改_config.yml中theme对应的内容, 如下，重启主题生效

Deploy to remote sites

向远程github_name.github.io仓库提交代码

Hexo Admin

安装hexo-admin, 次优雅地修改博客文章

安装以后, 重启hexo, 在本地启动输入 http://localhost:4000/admin/ 配置admin用户密码, 存入_config.yml中

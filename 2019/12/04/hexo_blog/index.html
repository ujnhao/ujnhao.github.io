Key Generate 配置github SSH key

以linux系统为例, 输入下面的命令并从.ssh目录下copy公钥(id_rsa.pub), 在Github仓库创建新的SSH KEY, 这样就可以向Github提交代码, 用Github账号名做仓库的前缀, 例如github_name.github.io

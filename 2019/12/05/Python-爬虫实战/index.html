requests

requests，Python HTTP 请求库

安装

发送请求

HTTP 请求方法我主要常用的是 GET、POST 方法。

请求返回 Response 对象，Response 对象是对 HTTP 协议中服务端返回给浏览器的响应数据的封装，响应的中的主要元素包括：状态码、原因、响应首部、响应 URL、响应encoding、响应体等等。

定制请求头

请求添加 HTTP 头部 Headers，需要传递一个 dict 作为 headers 关键字参数。

构建查询参数

GET请求参数

Requests 允许使用 params 关键字参数，以一个字典来提供，甚至可以是list。

POST请求参数

服务器要求发送的数据是表单数据，则可以指定关键字参数 data。

如果要求传递 json 格式字符串参数，则可以使用 json 关键字参数，参数的值都可以字典的形式传过去。

超时

如果没有显式指定了 timeout 值，requests 是不会自动进行超时处理的。如果遇到服务器没有响应的情况时，整个应用程序一直处于阻塞状态而没法处理其他请求。
